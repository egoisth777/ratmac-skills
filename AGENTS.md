# ratmac-skills — agent instructions

## Routing rule for markdown knowledge files

This project has a **single routing rule** for where markdown knowledge files
(notes, findings, concept docs, promoted snippets) may be written. It lives in
[`.arca/AGENTS.md`](.arca/AGENTS.md).

**Read [`.arca/AGENTS.md`](.arca/AGENTS.md) before writing, capturing, or
promoting any `.md` knowledge file.** It is the single source of truth for where
knowledge goes — and where it must NOT go (the scheduler tree, the store, or the
project source tree).
